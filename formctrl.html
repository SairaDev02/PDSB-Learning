Meters

Meters and progress bars are visual representations of numeric values

Uses <progress> and <meter> elements

Meter

Represents a fixed value delimited by max and min values

Lower part
Range between min and low, inclusive
Medium part
Range between low and high, exclusive
Higher part
Range between high and max, inclusive

optimum defines the optimum value for the meter element

Example:

75

Progress

Represents a value that changes overtime up to a maximum value specified by max attribute

Created using <progress> element

Example:

75/100
