Lists!

<ul> is used for unordered lists (no numbering)

  • Alpha
  • Beta
  • Charlie
  • Delta

<ul> is used for ordered lists (with numbering)

Includes <li> to apply order to list of items

  1. Alpha
  2. Bravo
  3. Charlie

Lists can be nested too

  1. Alpha
    • Alpha One
    • Alpha Two

Description!

Description consists of terms <dt> and descriptions <dd>

Chillet
Very adorable ice noodle
Astegon

Sarcastic Emphasis can be applied on words with <em> element

Important Emphasis can be applied on words using <strong> element
