Description Lists!

Description lists allows creation of a list of items or terms along with their descriptions

Example:

Apple
An apple is a fruit produced by various species of the genus Malus in the family Rosaceae, particularly Malus domestica. It has a round or oval shape and typically grows on trees. Apples have a tough outer skin that can be red, green, yellow, or combinations thereof, depending on the variety. The flesh inside is juicy and comes in shades of white to pale yellow.
Orange
An orange is a citrus fruit that grows on evergreen trees of the Citrus species, primarily Citrus sinensis. The fruit has a round or oval shape with a thick, bright-colored skin, which can be either smooth (as in Valencia and navel oranges) or rough (as in satsuma mandarins). Underneath the peel, an orange contains juicy segments of translucent pulp surrounded by membranes. Oranges are rich in vitamin C and have a sweet flavor with slightly tangy undertones.

Code Markup

There are a number of elements for marking up code in HTML

  • code: Markup generic code
  • pre: Markup code with whitespace retained
  • var: Markup variable names
  • kbd: Markup keyboard or other input into the computer
  • samp: Markup output of the code

Example:

            const para = document.querySelector('p');

            para.onclick = function() {
              alert('Owww, stop poking me!');
            }

You shouldn't use presentational elements like <font> and <center>.

In the above JavaScript example, para represents a paragraph element.
