Basic sections of a document

header:
A strip at the top where heading, logo, tagline, etc can be seen.
navigation bar:
Links to the site's main sections; represented by menu buttons, links, or tabs.
main:
An area that contains most of the unique content of a given webpage.
sidebar:
Peripheral info, links, quotes, ads, etc. Can be a secondary navigation system.
footer:
A strip at the bottom where generally contains fine print, copyright notices, or contact info

Markup Elements

  • header: <header>
  • navigation bar: <nav>
  • main content: <main>
    Contains article, section and div elements.
  • sidebar: <aside>; usually placed inside main
  • footer: <footer>
