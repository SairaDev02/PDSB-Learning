Basic Tables


Table element

<table> element contains the content of the table

Example:

Cell 1 Cell 2

<td> element corresponds to a table cell (table data) in a row


To put more rows, <tr> element is used to define rows. Use mulitple row elements for more rows displayed.

Example:

Cell 1 Cell 2 Cell 3
Cell 4 Cell 5 Cell 6

To add table headers, use <th> element

Example:

Header 1 Header 2 Header 3
Cell 1 Cell 2 Cell 3

To make a column or row to span, use colspan and rowspan and specify the value to span.

Example:

Animals
Hippopotamus
Horse Mare
Stallion
Crocodile
Chicken Hen
Rooster

To style the table and its contents, use <col> element within <colgroup> to style columns. Remove tables.css to show

Example:

Header 1 Header 2 Header 3
Cell 1 Cell 2 Cell 3
Cell 3 Cell 4 Cell 5
